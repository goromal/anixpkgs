# png

Generate PNG images from a variety of similar formats.


## Usage

```bash
usage: png inputfile outputfile

Create a png file.

Inputs:
    .png
    .gif
    .svg
    .jpeg
    .heic
    .tiff

Options:
    -r|--resize [e.g., 50%]  Resize the image.
    -s|--scrub               Scrub image metadata.

```
