# mp3unite

Unite mp3 files, much like with the `pdfunite` tool.


## Usage

```bash
usage: mp3unite [options] <MP3-sourcefile-1>..<MP3-sourcefile-n> <MP3-destfile>

Combine MP3 source files into a single destination MP3 file.

Options:
-h | --help     Print out the help documentation.
-v | --verbose  Print verbose output from ffmpeg

```
