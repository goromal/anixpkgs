# tasks_ui

Flask UI for task-tools on the ATS machine.

Provides a web interface for uploading tasks to Google Tasks using task-tools.
## Usage

```bash
usage: tasks_ui [-h] [--port PORT] [--subdomain SUBDOMAIN]
                [--spec-csv SPEC_CSV]

options:
  -h, --help            show this help message and exit
  --port PORT
  --subdomain SUBDOMAIN
  --spec-csv SPEC_CSV
```
